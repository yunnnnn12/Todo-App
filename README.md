# Todo App
- 프론트(React Native)와 백엔드(Spring Boot)로 구성된 Todo 애플리케이션
- 할 일 CRUD 및 완료 상태 관리 제공

## ☑️ Architecture
- Frontend: React Native
- Backend: Spring Boot
- Database: MySQL

## ☑️ Project Structure
- Todo - App 구조 :
    - frontend # React Native app
    - backend # Spring Boot API server
    - README.md # 프로젝트 전체 설명

## ☑️ Features
- 할 일 목록 조회
- 할 일 추가 / 삭제
- 할 일 완료 처리
- 프론트엔드 <-> 백엔드 REST API 연동
- Azure에 배포된 백엔드 서버와 통신하여 실제 서비스 환경에서 동작

## ☑️ DevOps & Deployment
- Docker를 활용해 개발 및 실행 환경 컨테이너화
- Azure를 사용해 백엔드 애플리케이션 배포

## ☑️ Git & Collaboration
- 기능 단위로 커밋을 분리하여 변경 이력 관리
- Conventional Commits 규칙 기반으로 커밋 메세지 작성
